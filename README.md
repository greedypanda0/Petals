# 🌸 Petals

**An AI chatbot for loners.**
Because sometimes, talking to a machine is better than getting left on *read*.

---

## 📅 Clone & Setup

```bash
git clone https://github.com/greedypanda0/Petals
cd petals/
```

---

## 🛠️ Environment Variables

Create a `.env` file and fill in the following:

```env
DATABASE_URL=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

NEXTAUTH_SECRET=
NEXTAUTH_URL=

NODE_ENV=development
```

---

## 🚀 Run It

```bash
npm install
npm run dev
```
